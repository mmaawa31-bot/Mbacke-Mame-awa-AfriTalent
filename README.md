## Description du projet
AfriTalent est une plateforme qui met en relation les clients avec des freelances africains. Le site propose un système de filtrage, un dark mode, et des animations au scroll.

## Fonctionnalités implémentées
- **HTML/CSS/JS** : Structure responsive + animations fade-in
- **Dark mode** : Sauvegarde avec localStorage  
- **Filtrage dynamique** : Recherche de freelances par catégorie
- **Compteurs animés** : Animation au scroll
- **Validation formulaire** : Page contact
- **Design Bento Grid** : Section hero et catégories
## Technologies utilisées
HTML5, CSS3, JavaScript vanilla, GitHub Pages
